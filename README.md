# firechat

Веб-приложение, опубликованное через GitHub Pages.

## 🚀 Как открыть

После публикации на GitHub Pages сайт будет доступен по адресу:  
`https://ВАШ_ЛОГИН.github.io/firechat`

## 📖 Инструкция по публикации

1. Создай репозиторий `firechat` на github.com
2. Загрузи все файлы из этого архива в репозиторий
3. Перейди в `Settings → Pages`
4. Выбери ветку `main` и папку `/ (root)`
5. Нажми **Save** — сайт появится через 1-2 минуты! 🎉
